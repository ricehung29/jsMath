# pyAdMath
## Finding power's last digit

### Example 1 :
If the last digit of 2022 to the power 9 is c, find the value of c.

Python Code :
```
print(pow(2022,9)%10)
```

Output :
`2`

### Example 2 :
If the last digit of 2022 to the power 2022 is c, find the value of c.

Python Code :
```
print(pow(2022,2022)%10)
```

Output :
`4`
